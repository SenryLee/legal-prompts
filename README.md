# 法律工作提示词合集

> 专为法律从业者设计的 AI 提示词工具库，覆盖诉讼、合同、非诉、劳动、公司治理、尽调与合规等高频法律工作场景

## 📖 项目简介

本提示词合集是为法律从业者（律师、法官助理、法务等）量身定制的 AI 辅助工具库，覆盖**诉讼案件处理**、**合同审查**、**非诉法律服务**、**劳动人事**、**公司治理**、**交易与尽职调查**、**企业合规**等核心工作场景。

每个提示词都经过精心设计，包含：
- ✅ **专业角色设定**：资深律师、法官、法官助理、研究员等
- ✅ **立场确认机制**：自动识别用户立场（原告/被告、甲方/乙方等）
- ✅ **时间与事件维度**：案件必含时间线和事件梳理
- ✅ **事实问题整理**：详细、清晰、无遗漏的事实分析
- ✅ **结构化输出**：清晰的结论和多维度分析

## 🎯 设计原则

### 1. 立场明确
每个提示词都会先确认用户的立场（原告/被告、甲方/乙方等），确保分析角度正确。

### 2. 事实为基
强调对事实的详细梳理，特别是时间线和事件链，确保事实清晰、无遗漏。

### 3. 逻辑严密
每个提示词都有清晰的分析框架和逻辑链条，确保输出结构化、有深度。

### 4. 结论明确
输出的结论清晰、辩证、多维度，直接指导实际工作。

### 5. 专业导向
角色设定涵盖资深律师、法官、法官助理、研究员等专业角色。
