# 法律工作提示词合集

> 面向律师、法务、法官助理、法律研究员等法律从业者的中文 AI 提示词库，覆盖诉讼、合同、非诉、劳动、公司治理、交易尽调、企业合规与知识产权等高频工作场景。

## 项目简介

本仓库用于沉淀可以直接复制使用的法律工作提示词。每个提示词都尽量保持以下共同结构：

* 明确任务目标
* 设定可信的法律角色
* 要求用户先补齐立场与核心材料
* 按时间线、事实矩阵、风险矩阵展开分析
* 输出可直接用于诉讼、谈判、审查、调查或决策支持的结构化结果

当前仓库共包含 **222 个可直接使用的 Markdown 提示词文件**：

* **219 个**已纳入八大法律分类目录
* **3 个**位于根目录，作为辅助型提示词使用

## 分类说明

### 诉讼案件处理

目前已经形成完整的诉讼工具链，覆盖：

* 案卷阅读与事实整理：证据真实性审查、对方证据弱点识别、案件时间线整理、多被告责任分担
* 诉讼策略规划：可诉性评估、路径选择、追加第三人、反诉可行性、成本收益分析
* 起诉、答辩、上诉、再审、监督、反诉、仲裁等文书生成
* 庭前会议、质证、发问、鉴定、证人询问、庭审提纲、法庭辩论等庭审应对
* 财产保全、证据保全、行为保全、调查令、鉴定、管辖异议、回避、延期开庭等程序申请
* 执行、执行异议、追加被执行人、终本恢复、拒执线索、执行和解、以物抵债等执行场景
* 诉讼方案、和解调解、诉讼时效、撤诉、二审新证据、仲裁协议效力等策略场景

### 合同审查

目前覆盖四层结构：

* 基础审查：主体资格、效力判断、格式条款、语言规范、谈判策略、履行监控、风险替换、版本对比
* 合同文本审查：识别基础风险、缺失条款、表达问题、权利义务失衡
* 合同修改建议：生成修改意见、提炼谈判要点
* 专项审查：围绕解除终止、违约责任、付款验收、担保、知识产权、SaaS、经销代理、Term Sheet、建设工程、房屋租赁、借款、买卖、劳动、技术、特许经营、对赌、VIE、争议解决、不可抗力等高频条款和文件

### 非诉法律服务

适合处理：

* 口头法律咨询、法律问题初步分析、法律风险评估、合规差距分析
* 交易可行性分析、争议解决方案比选
* 律师函、法律意见书、催款函、声明函、公告通知、授权委托书
* 见证法律意见书、专项法律备忘录、确认函/回函、承诺函/保证函
* 情况说明/证明、会议纪要、内部培训材料、客户汇报PPT、项目总结、法律检索报告
* 商账催收与回款谈判

### 劳动人事

## 编号与排序规则

* 同一子目录内统一采用 `01-`、`02-`、`03-` 的两位数序号格式
* 优先按实务使用顺序、工作流顺序和阅读便利性排序

## 使用建议

1. 先定位最接近的业务场景目录。
2. 打开对应提示词后，优先补齐立场、核心材料、关键时间线和目标。
3. 将提示词直接投喂给模型时，尽量一并提供原始证据、文书、纪要、往来记录或合同文本。
4. 如果现有目录仍缺少某个高频场景，可先记录到 [待补充优先级清单.md](./待补充优先级清单.md) 再继续扩充。
