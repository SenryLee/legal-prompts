# 法律工作提示词合集

> 面向律师、法务、法官助理、法律研究员等法律从业者的中文 AI 提示词库，覆盖诉讼、合同、非诉、劳动、公司治理、交易尽调与企业合规等高频工作场景。

## 项目简介

本仓库用于沉淀可以直接复制使用的法律工作提示词。每个提示词都尽量保持以下共同结构：

- 明确任务目标
- 设定可信的法律角色
- 要求用户先补齐立场与核心材料
- 按时间线、事实矩阵、风险矩阵展开分析
- 输出可直接用于诉讼、谈判、审查、调查或决策支持的结构化结果

当前仓库共包含 **96 个可直接使用的 Markdown 提示词文件**：

- **93 个**已纳入七大法律分类目录
- **3 个**位于根目录，作为辅助型提示词使用

### 本次更新解决了什么问题

这次整合主要补齐了原库中几个比较明显的空白：

- 解决了诉讼类提示词偏重通用文书、但对程序申请、庭审细节、执行攻防覆盖不够的问题。

- 解决了合同专项审查场景不够细、对 SaaS、经销代理、Term Sheet 等实务高频交易文件覆盖不足的问题。
- 解决了劳动人事模块对组织调整场景覆盖不够，缺少调岗降薪、批量裁员等用工优化场景的问题。
- 解决了企业合规模块偏重制度和文本审查、但对内部调查和监管应对覆盖不足的问题。
- 解决了尽调场景缺少“红旗问题分级”和交易推进优先级视角的问题。
- 解决了公司治理与非诉模块中，对控制权争议和商账催收谈判缺少直接可用提示词的问题。

## 分类说明

### 诉讼案件处理

目前已经形成相对完整的诉讼工具链，覆盖：

- 案卷阅读与事实整理
- 起诉、答辩、上诉、再审、监督、反诉、仲裁等文书生成
- 庭前会议、质证、发问、鉴定、证人询问等庭审应对
- 财产保全、证据保全、行为保全、调查令、鉴定、管辖异议等程序申请
- 执行、执行异议、追加被执行人、终本恢复、拒执线索等执行场景
- 诉讼方案、和解调解、诉讼时效、撤诉、二审新证据等策略场景

### 合同审查

目前覆盖三层结构：

- 合同文本审查：识别基础风险、缺失条款、表达问题、权利义务失衡
- 合同修改建议：生成修改意见、提炼谈判要点
- 专项审查：围绕解除终止、违约责任、付款验收、担保、知识产权、SaaS、经销代理、Term Sheet 等高频条款和文件

### 非诉法律服务

适合处理：

- 书面法律咨询答复
- 律师函与法律意见书
- 商账催收与回款谈判

### 劳动人事

覆盖从制度到争议的常见链路：

- 解除劳动合同
- 不胜任解除
- 调岗降薪
- 批量裁员
- 员工违纪调查
- 劳动仲裁申请
- 竞业限制协议
- 员工手册合法性审查

### 公司治理

目前包含：

- 股东会、董事会决议审查与起草
- 股东争议与控制权稳定
- 对外担保专项审查
- 关联交易专项审查

### 交易与尽职调查

目前包含：

- 法律尽职调查风险摘要
- 尽调红旗问题分级报告
- 股权转让协议专项审查
- 投融资条款风险识别

### 企业合规

覆盖：

- 合规风险排查与整改建议
- 数据合规专项排查
- 隐私政策与用户协议审查
- 商业秘密保护方案设计
- 广告宣传合规审查
- 平台规则专项审查
- 内部举报调查方案设计
- 行政监管问询回复方案

## 使用建议

1. 先定位最接近的业务场景目录。
2. 打开对应提示词后，优先补齐立场、核心材料、关键时间线和目标。
3. 将提示词直接投喂给模型时，尽量一并提供原始证据、文书、纪要、往来记录或合同文本。
4. 如果现有目录仍缺少某个高频场景，可先记录到 [待补充优先级清单.md](./待补充优先级清单.md) 再继续扩充。
