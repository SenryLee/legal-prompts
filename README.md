# 法律工作提示词合集

> 面向律师、法务、法官助理、法律研究员等法律从业者的中文提示词库。  
> **结构原则：诉讼与非诉分开**——先选大类，再选子类。

## 项目简介

本仓库用于沉淀可直接复制使用的中文法律工作提示词，覆盖诉讼案件处理、合同审查、非诉文书、劳动人事、公司治理、交易与尽职调查、企业合规、知识产权等高频场景。

提示词统一骨架（详见 [`docs/PROMPT_SPEC.md`](docs/PROMPT_SPEC.md)）一般包括：

* 明确任务目标
* 设定可信的法律角色（含场景化立场）
* 说明输入要求
* 按场景给出分析框架
* 输出可直接用于诉讼、谈判、审查、调查或决策支持的结构化结果

## 诉讼与非诉如何区分

| 任务类型 | 目录 |

| 行政复议、行政诉讼、行政应诉 | [`诉讼/11-行政诉讼与复议/`](诉讼/11-行政诉讼与复议/) |
| 破产申请、重整计划、预重整、债权人会议 | [`诉讼/12-破产重整/`](诉讼/12-破产重整/) |

| 审合同、改合同、谈条款 | [`非诉/01-合同审查/`](非诉/01-合同审查/) |
| 书面咨询、风险评估、方案比选 | [`非诉/02-咨询答复/`](非诉/02-咨询答复/) |

| 章程、决议、股权激励、股东知情权（诉前） | [`非诉/05-公司治理/`](非诉/05-公司治理/) |
| 尽调、交易文件、对赌回购条款审查 | [`非诉/06-交易与尽职调查/`](非诉/06-交易与尽职调查/) |
| 数据、反垄断、出口管制、披露等专项合规 | [`非诉/07-企业合规/`](非诉/07-企业合规/) |

经验法则：已经要立案、开庭、裁决、执行或破产受理的，进 **诉讼**；仍在审查、设计、谈判或合规整改的，进 **非诉**。

## 怎么选提示词

### 1. 先选诉讼还是非诉

见上表，或打开 [`诉讼/README.md`](诉讼/README.md)、[`非诉/README.md`](非诉/README.md) 做二次导航。

### 2. 再按手上材料缩小范围

* 案卷、证据、判决、对方诉讼文书 → 诉讼
* 合同文本、制度、尽调资料、合规清单 → 非诉

### 3. 再按期望输出选子目录

* 风险扫描、问题诊断 → 基础审查、文本问题诊断、咨询答复、尽调
* 正式文书 → `诉讼/06-文书生成` 或 `非诉/03-非诉文书`
* 谈判与修改 → 合同修改与谈判、交易谈判、和解方案
* 程序动作 → `诉讼/05-程序申请`、`09-执行程序`、`11-行政诉讼与复议`

### 4. 使用前确认立场与法域

* 代表谁：原告/被告、甲方/乙方、投资人/融资方、用人单位/劳动者等
* 要什么：结论、风险排序、文书草稿、修改建议、谈判策略、执行路径
* 默认适用中国大陆法律；涉及港澳台或涉外的，须另行声明准据法与管辖

### 5. 常见重叠主题怎么选

| 主题 | 优先使用 | 说明 |
|------|----------|------|
| 关联交易（治理程序/决议） | `非诉/05-公司治理/专项审查/` | 决议、回避、公允性论证 |
| 关联交易（监管披露/问询） | `非诉/07-企业合规/专项合规/` | 上市披露与监管口径 |
| 股东协议（日常治理） | `非诉/05-公司治理/内部治理/` | 起草与治理安排 |
| 股东协议（投融资交易） | `非诉/06-交易与尽职调查/交易专项审查/` | 交易谈判审查 |
| 知识产权尽调（专业全案） | `非诉/08-知识产权/综合/` | 买方/卖方知产尽调 |
| 知识产权尽调（交易配合） | `非诉/06-交易与尽职调查/尽职调查/` | 交易项目中的 IP 配合清单 |
| 数据合规（一般排查） | `非诉/07-企业合规/专项合规/02-...` | 收集、共享、安全等 |
| 数据出境 | `非诉/07-企业合规/专项合规/23-...` | 跨境传输路径与材料 |
| 劳动用工制度/解除评估 | `非诉/04-劳动人事/` | 尚未进入仲裁 |
| 劳动仲裁文书与程序 | `诉讼/10-劳动争议仲裁/` | 申请、答辩、举证、调解 |

## 分类说明

### 诉讼

覆盖证据与事实、案卷分析、策略规划、程序申请、文书生成、庭审应对、裁判分析、执行，以及劳动争议仲裁、行政诉讼与复议、破产重整、刑民交叉等场景。

### 非诉

* **合同审查**：基础审查、文本诊断、修改谈判、条款专题、合同类型、商业与交易场景
* **咨询答复 / 非诉文书**：咨询意见、律师函、法律意见书、内部材料与汇报
* **劳动人事**：专项处理与制度审查
* **公司治理**：内部治理与担保、关联交易等专项审查
* **交易与尽职调查**：尽调、交易文件、交易专项审查
* **企业合规**：数据、广告、反垄断、出口管制、证券披露等专项合规
* **知识产权**：商标、专利、著作权及综合管理

## 使用建议

* 先按诉讼/非诉定位，再按二级目录选择最接近的任务
* 实际使用前补齐立场、核心材料、目标和约束条件
* 诉讼、合同、尽调、合规类任务，优先保留时间线、风险矩阵、优先级清单等结构化输出
