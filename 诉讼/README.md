# 诉讼业务提示词

本目录收录**已进入或准备进入争议解决程序**的提示词：民事诉讼、商事仲裁衔接、劳动仲裁、行政诉讼与复议、执行、破产重整、刑民交叉等。

非争议性的审查、咨询、合规、交易文件请到 [`../非诉/`](../非诉/)。

## 选用顺序（简图）

1. 事实与证据不清 → `01` / `02`
2. 决定打不打、怎么打 → `03` / `04`
3. 程序动作 → `05`
4. 出文书 → `06`
5. 临近开庭 → `07`
6. 分析结果/准备上诉 → `08` + `04`
7. 胜诉后回款 → `09`
8. 劳动仲裁专项 → `10`
9. 行政争讼 → `11`
10. 破产重整 → `12`
11. 涉刑风险 → `13`

## 与非诉的交界

| 场景 | 诉讼 | 非诉 |
|------|------|------|
| 劳动争议 | `10-劳动争议仲裁` | `非诉/04-劳动人事`（解除评估、制度、竞业审查） |
| 监管处罚 | `11-行政诉讼与复议` | `非诉/07-企业合规`（问询回复、专项合规） |
| 对赌回购争议已开战 | `04`/`06`/`13` | `非诉/06/.../09-10`（条款审查与诉前谈判） |
| 知产侵权诉讼 | `06` 文书 + 策略 | `非诉/08-知识产权`（侵权分析、无效、许可） |
