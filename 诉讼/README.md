# 诉讼业务提示词

本目录收录已进入或准备进入争议解决程序的提示词，包括民事诉讼、劳动仲裁、行政诉讼与复议、执行、破产重整、刑民交叉等。

审查、咨询、合规、交易文件等非争议事项，请到 [`../非诉/`](../非诉/)。

当前共 **88** 个提示词。

| 编号 | 子类 | 数量 | 适用场景 |

## 选用顺序

1. 事实与证据不清 → `01` / `02`
2. 决定是否起诉、如何起诉 → `03` / `04`
3. 需要程序动作 → `05`
4. 起草诉讼文书 → `06`
5. 临近开庭 → `07`
6. 分析裁判结果或准备上诉 → `08` + `04`
7. 胜诉后推进回款 → `09`
8. 劳动仲裁 → `10`
9. 行政争讼 → `11`
10. 破产重整 → `12`
11. 涉及刑事风险 → `13`

## 与非诉的交界

| 场景 | 本目录 | 非诉目录 |
|------|--------|----------|
| 劳动争议 | `10-劳动争议仲裁` | `非诉/04-劳动人事`（解除评估、制度、竞业审查） |
| 监管处罚 | `11-行政诉讼与复议` | `非诉/07-企业合规`（问询回复、专项合规） |
| 对赌回购已进入争议程序 | `04` / `06` / `13` | `非诉/06-交易与尽职调查`（条款审查与诉前谈判） |
| 知识产权侵权诉讼 | `06` 文书及策略类 | `非诉/08-知识产权`（侵权分析、无效、许可） |
