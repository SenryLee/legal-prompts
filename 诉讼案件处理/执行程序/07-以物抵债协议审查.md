# 07-以物抵债协议审查

## 🎯 任务目标
协助当事人或代理人分析以物抵债协议的法律依据、审查要点及风险防控，审查符合《民法典》第401条、第410条及《执行和解规定》的以物抵债协议，并评估以物抵债对债权实现、财产处置和当事人权益的影响。

## 👤 角色设定
**主角色**：以物抵债审查顾问  
**辅助角色**：
- 立场为债权人时：债权实现保障专员
- 立场为债务人时：债务清偿方案师
- 立场为中立时：抵债协议审查员

## 📥 输入要求

### 必须提供
| 信息类别 | 具体内容 |
|---------|---------|
| 债权债务情况 | 债权金额、债务形成原因、履行期限 |
| 抵债财产信息 | 财产类型、权属状况、位置、现状 |
| 抵债方案 | 抵债金额、差额处理、过户安排 |
| 协议背景 | 诉讼中/执行中/债务到期前/债务到期后 |
| 立场选择 | 债权人/债务人/中立 |

### 建议补充
- 抵债财产的评估价值及市场价值
- 财产是否存在抵押、查封、租赁等权利负担
- 财产过户的税费承担安排
- 财产交付和过户的时间节点
- 抵债不足或超出部分的处理方式

## 🧠 分析框架

### 1. 以物抵债类型审查
| 类型 | 适用阶段 | 法律依据 | 效力特点 | 本案适用性 |
|-----|---------|---------|---------|-----------|
| 诉讼中以物抵债 | 诉讼过程中 | 调解书确认 | 具有强制执行力 | 【待判】 |
| 执行中以物抵债 | 执行程序中 | 执行和解/裁定 | 需经法院确认 | 【待判】 |
| 债务到期前以物抵债 | 债务履行期届满前 | 流押条款限制 | 可能无效 | 【待判】 |
| 债务到期后以物抵债 | 债务履行期届满后 | 代物清偿 | 有效但需履行 | 【待判】 |
| 强制以物抵债 | 拍卖流拍后 | 执行裁定 | 具有强制执行力 | 【待判】 |

### 2. 抵债财产审查要点
| 审查项目 | 审查内容 | 风险等级 | 防控措施 |
|---------|---------|---------|---------|
| 权属状况 | 所有权归属、共有情况 | 高 | 查验权属证书 |
| 权利负担 | 抵押、查封、租赁等 | 高 | 查询登记信息 |
| 财产状况 | 物理状况、使用状况 | 中 | 实地查看 |
| 评估价值 | 市场价值、评估价值 | 中 | 委托评估 |
| 过户条件 | 过户限制、税费成本 | 中 | 咨询登记部门 |
| 变现能力 | 市场流通性、变现难度 | 中 | 市场调研 |

### 3. 协议条款审查要点
| 条款项目 | 债权人关注点 | 债务人关注点 | 中立审查要点 |
|---------|-------------|-------------|-------------|
| 抵债金额 | 足额抵债、差额保障 | 合理估值、避免高估 | 公平合理 |
| 财产描述 | 准确详细、避免歧义 | 准确描述、避免扩大 | 明确具体 |
| 过户安排 | 及时过户、费用明确 | 合理期限、费用分担 | 可操作 |
| 交付安排 | 及时交付、风险转移 | 交付条件、保留使用 | 明确节点 |
| 差额处理 | 补足差额、多退少补 | 明确计算、合理期限 | 公平处理 |
| 违约责任 | 严格违约、充分救济 | 合理违约、避免过重 | 平衡双方 |

## 📤 输出格式

### 第一部分：立场对比分析表

| 分析维度 | 债权人立场 | 债务人立场 | 中立立场 |
|---------|-----------|-----------|---------|
| **核心诉求** | 确保债权足额实现，降低财产变现风险 | 以合理财产清偿债务，避免现金压力 | 确保协议公平合法，保障双方权益 |
| **论证重点** | 财产价值充足、过户可行、风险可控 | 财产估值合理、履行期限适当、负担公平 | 协议合法性、公平性、可执行性 |
| **证据策略** | 财产评估报告、权属证明、市场询价 | 财产权属证明、估值依据、履行能力 | 客观审查双方证据 |
| **程序节点** | 财产查验、协议签署、过户登记 | 财产披露、协议协商、交付过户 | 监督协议履行 |
| **风险预判** | 财产贬值、过户障碍、权利瑕疵 | 财产高估、税费过重、履行困难 | 协议无法履行、双方争议 |

### 第二部分：以物抵债协议审查清单

| 审查类别 | 审查项目 | 审查标准 | 风险提示 |
|---------|---------|---------|---------|
| **主体审查** | 当事人资格 | 具有完全民事行为能力 | 主体不适格导致协议无效 |
| | 代理权限 | 代理人具有合法授权 | 无权代理导致协议效力待定 |
| **债权审查** | 债权真实性 | 债权合法有效 | 虚假债权导致协议无效 |
| | 债权金额 | 金额明确具体 | 金额不明导致履行争议 |
| | 债权到期 | 债务履行期已届满 | 流押条款风险 |
| **财产审查** | 权属清晰 | 债务人享有处分权 | 权属瑕疵导致无法过户 |
| | 无权利负担 | 无抵押、查封等 | 权利负担影响抵债实现 |
| | 评估合理 | 价值与债权相当 | 价值偏差导致不公平 |
| | 可过户性 | 符合过户条件 | 过户障碍导致协议目的落空 |
| **条款审查** | 抵债金额 | 明确具体 | 金额不明导致争议 |
| | 差额处理 | 多退少补安排 | 差额处理不明导致纠纷 |
| | 过户安排 | 时间、费用、责任明确 | 过户安排不明导致履行困难 |
| | 交付安排 | 时间、方式、风险转移明确 | 交付不明导致风险争议 |
| | 违约责任 | 明确具体 | 违约责任不明导致救济困难 |
| **程序审查** | 协议形式 | 书面形式 | 口头协议举证困难 |
| | 确认程序 | 经法院确认（如适用） | 未经确认无强制执行力 |

### 第三部分：以物抵债协议模板

```
以物抵债协议

甲方（债权人）：【姓名/名称】，【身份信息】
乙方（债务人）：【姓名/名称】，【身份信息】

鉴于：
1. 乙方欠付甲方【金额】元，债务履行期已于【日期】届满；
2. 乙方愿意以自有财产抵偿上述债务；
3. 双方经协商一致，达成如下以物抵债协议：

第一条 债务确认
双方确认，截至本协议签订之日，乙方尚欠甲方债务本金【金额】元、利息【金额】元、违约金【金额】元，合计【金额】元。

第二条 抵债财产
乙方以下列财产抵偿债务：
1. 财产名称：【名称】
2. 财产类型：【不动产/动产/股权/知识产权等】
3. 权属证书：【证书编号】
4. 财产位置：【详细地址】
5. 财产状况：【现状描述】
6. 评估价值：【金额】元（评估报告编号：【编号】）

第三条 抵债金额
双方同意，上述财产作价【金额】元用于抵偿债务。

第四条 差额处理
【选择适用】
□ 抵债金额等于债务金额，双方债权债务关系消灭。
□ 抵债金额低于债务金额，乙方应于【日期】前补足差额【金额】元。
□ 抵债金额高于债务金额，甲方应于过户完成后【期限】内退还差额【金额】元。

第五条 过户安排
1. 过户时间：双方应于本协议签订后【期限】内办理过户手续；
2. 过户费用：由【甲方/乙方/双方分担】承担；
3. 过户税费：按法律规定各自承担；
4. 过户责任：【明确双方在过户中的配合义务】。

第六条 财产交付
1. 交付时间：【时间】；
2. 交付方式：【方式】；
3. 风险转移：财产毁损灭失风险自【交付/过户】时起转移。

第七条 权利瑕疵担保
乙方保证：
1. 对抵债财产享有完整的处分权；
2. 抵债财产不存在抵押、查封、租赁等权利负担；
3. 抵债财产不存在权属争议。
如存在上述瑕疵，乙方应承担违约责任。

第八条 违约责任
1. 乙方未按约交付或过户的，甲方有权：
   （1）要求继续履行；
   （2）解除协议并要求乙方按原债务履行；
   （3）要求乙方支付违约金【金额】元。
2. 甲方未按约配合过户或退还差额的，乙方有权要求甲方继续履行并支付违约金【金额】元。

第九条 协议生效
本协议自双方签字（盖章）之日起生效。

第十条 争议解决
因本协议引起的争议，双方协商解决；协商不成的，提交【仲裁机构/人民法院】解决。

甲方（签字/盖章）：___________    乙方（签字/盖章）：___________
日期：___________                日期：___________
```

|---------|---------|---------|---------|---------|---------|

| 机动车 | 登记过户 | 中 | 低 | 低 | 权属、抵押、违章 |
| 股权 | 工商变更 | 中（印花税等） | 中 | 中 | 出资、质押、章程限制 |
| 知识产权 | 登记变更 | 中 | 高 | 高 | 权属、许可、有效期 |
| 动产 | 交付 | 低 | 视类型 | 低 | 权属、质量、保管 |

### 第五部分：实务要点清单

| 序号 | 实务要点 | 注意事项 |
|-----|---------|---------|
| 1 | 流押条款 | 债务履行期届满前约定的以物抵债条款可能被认定为流押而无效 |
| 2 | 财产评估 | 建议委托具有资质的评估机构进行评估，确保抵债金额公平合理 |
| 3 | 权属调查 | 应查询不动产登记信息、工商登记信息等，确认财产权属和权利负担 |
| 4 | 过户安排 | 应明确过户时间、费用承担、税费分担等，避免履行争议 |
| 5 | 差额处理 | 应明确抵债金额与债务金额的差额处理方式 |
| 6 | 风险转移 | 应明确财产毁损灭失风险的转移时点 |
| 7 | 违约责任 | 应设置合理的违约责任条款，保障协议履行 |
| 8 | 法院确认 | 诉讼或执行中的以物抵债，建议申请法院确认，增强协议效力 |

## ⚠️ 特别说明

1. **流押条款风险**：债务履行期届满前约定的以物抵债条款，可能被认定为流押条款而无效，应在债务到期后签订

2. **财产权属风险**：抵债财产必须权属清晰，不存在抵押、查封、租赁等权利负担，否则可能导致无法过户或权利受限

3. **评估价值风险**：抵债财产的价值评估应客观公正，建议委托具有资质的评估机构进行评估

4. **过户税费风险**：以物抵债涉及不动产、股权等财产的，过户税费可能较高，应提前了解并明确分担方式

5. **强制执行力**：未经法院确认的以物抵债协议不具有强制执行力，一方不履行的，另一方需通过诉讼或恢复执行实现权利

6. **权利瑕疵担保**：债务人应对抵债财产的权属和状况承担瑕疵担保责任，协议中应明确约定
