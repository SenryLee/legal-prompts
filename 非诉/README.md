# 非诉业务提示词

本目录收录审查、咨询、合规、交易、治理、制度与非争议性文书类提示词。

已进入诉讼、仲裁、执行、行政争讼、破产司法程序，或需要刑民交叉程序安排的，请到 [`../诉讼/`](../诉讼/)。

当前共 **160** 个提示词。

| 编号 | 模块 | 数量 | 主要子类 |
|------|------|------|----------|
| 01 | 合同审查 | 48 | 基础审查、文本问题诊断、修改与谈判、条款专题、合同类型、商业与交易场景 |
| 02 | 咨询答复 | 7 | 书面/口头咨询、风险评估、差距分析、可行性、争议方案比选 |
| 03 | 非诉文书 | 17 | 律师函、意见书、催款、声明、备忘录、培训与汇报等 |
| 04 | 劳动人事 | 21 | 专项处理、制度审查（劳动仲裁文书见诉讼/10） |
| 05 | 公司治理 | 11 | 内部治理、专项审查（担保/关联交易等） |
| 06 | 交易与尽职调查 | 20 | 尽职调查、交易文件、交易专项审查 |

## 合同审查怎么选

1. 先看整体基础风险 → `基础审查/`
2. 文本本身的问题 → `文本问题诊断/`
3. 需要修改意见或谈判要点 → `修改与谈判/`
4. 关注某类共性条款 → `条款专题/`
5. 典型合同类型 → `合同类型/`（含融资租赁、特许经营等）
6. 特定行业或商业模式 → `商业与交易场景/`

## 选用顺序

1. 合同或协议文本 → `01`
2. 客户提问、要结论或方案比选 → `02`
3. 律师函、意见书、对内材料 → `03`
4. 用工与制度 → `04`（若已进入仲裁 → `诉讼/10`）
5. 章程、决议、股权治理 → `05`
6. 投融资、并购、尽调 → `06`
7. 监管与专项合规 → `07`
8. 商标、专利、著作权 → `08`
