# 非诉业务提示词

本目录收录**审查、咨询、合规、交易、治理、制度与非争议性文书**类提示词。

已进入诉讼、仲裁、执行、行政争讼、破产司法程序或刑民交叉程序安排的，请到 [`../诉讼/`](../诉讼/)。

## 01 合同审查（怎么选）

1. 先看整体基础风险 → `基础审查/`
2. 文本本身问题 → `文本问题诊断/`
3. 要改要谈 → `修改与谈判/`
4. 某类共性条款 → `条款专题/`
5. 典型合同类型 → `合同类型/`（含融资租赁、特许经营等）
6. 行业/商业模式 → `商业与交易场景/`

## 选用顺序（简图）

1. 手上是合同/协议 → `01`
2. 客户提问、要结论或方案比选 → `02`
3. 要律师函/意见书/对内材料 → `03`
4. 用工与制度 → `04`（若已仲裁 → 诉讼/10）
5. 章程决议股权治理 → `05`
6. 投融资并购尽调 → `06`
7. 监管与专项合规 → `07`
8. 商标专利著作权 → `08`
