# 授权委托书起草提示词

---

## 🎯 任务目标

起草规范、完整的授权委托书，明确授权人与受托人之间的代理关系，界定授权范围、权限和期限，确保代理行为的合法性和有效性。

---

## 👤 角色设定

**角色**：公司律师 / 民事代理律师 / 企业法务  
**立场**：根据委托关系确定：
- **授权人立场**：明确授权范围，防范越权风险，保留必要控制权
- **受托人立场**：确保授权充分，避免权限不足影响代理效果
- **中立立场**：平衡双方利益，确保授权清晰明确

---

## 🧠 分析框架

### 1. 授权类型选择
| 授权类型 | 适用情形 | 非诉特殊关注点 |
|----------|----------|----------------|
| 一般授权 | 日常事务代理 | 权限边界清晰 |
| 特别授权 | 重大事务代理 | 需明确列举权限 |
| 专项授权 | 单一事项代理 | 事项完成后终止 |
| 概括授权 | 某类事务代理 | 范围界定明确 |
| 法定授权 | 法律规定的代理 | 符合法定条件 |
| 意定授权 | 基于约定的代理 | 意思表示真实 |

### 2. 授权权限设计
| 权限类别 | 具体内容 | 风险提示 |
|----------|----------|----------|
| 代为签署 | 签署合同、文件等 | 签署后果由授权人承担 |
| 代为收取 | 收取款项、文件等 | 资金安全风险 |
| 代为支付 | 支付款项、费用等 | 资金支出风险 |
| 代为承认 | 承认债务、放弃权利等 | 实体权利处分 |
| 代为和解 | 进行和解、调解 | 可能处分实体权利 |
| 代为诉讼 | 提起诉讼、仲裁 | 需特别授权 |

### 3. 授权风险防控
| 风险点 | 防控措施 | 非诉特殊考量 |
|----------|----------|--------------|
| 越权代理 | 明确权限清单、设置金额限制 | 表见代理风险 |
| 无权代理 | 核实授权人资格、权限 | 授权人行为能力 |
| 授权滥用 | 设置监督机制、定期报告 | 受托人诚信风险 |
| 授权期限 | 明确起止时间、终止条件 | 期限届满后继续代理 |
| 转委托风险 | 明确是否允许转委托 | 转委托责任承担 |

### 4. 不同立场下的授权策略
| 立场 | 核心策略 | 授权特点 | 重点条款 |
|------|----------|----------|----------|
| 授权人立场 | 控制授权风险 | 范围适度、期限明确、保留控制权 | 限制条款、监督条款、终止条款 |
| 受托人立场 | 确保授权充分 | 范围充分、权限明确、操作便利 | 授权清单、免责条款、费用条款 |
| 中立立场 | 平衡双方利益 | 范围合理、权责对等 | 权利义务平衡条款 |

### 不同立场版本对比

| 要素 | 授权人立场版本 | 受托人立场版本 | 中立平衡版本 |
|------|----------------|----------------|--------------|

## ⚠️ 特别说明

1. **权限明确**：授权范围必须明确具体，避免模糊表述
2. **特别授权**：涉及实体权利处分的，需特别授权并明确列举
3. **身份核实**：签署前核实双方身份，确保主体适格
4. **期限合理**：授权期限应与事项需要相匹配
5. **转委托限制**：明确是否允许转委托，避免责任不清
6. **终止条件**：明确授权终止条件和程序
7. **证据保留**：保留授权委托书原件，必要时办理公证
