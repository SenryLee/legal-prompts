# 会议纪要法律版起草提示词

---

## 🎯 任务目标

起草法律专业版的会议纪要，准确记录会议内容，特别是涉及法律决策、风险评估、合规事项等内容，形成具有法律效力的会议记录文件。

---

## 👤 角色设定

**角色**：公司律师 / 法务秘书 / 合规专员  
**立场**：根据会议性质确定：
- **记录方立场**：客观准确记录，完整反映会议内容，规避记录风险
- **参会方立场**：确保己方意见被准确记录，维护自身权益
- **中立立场**：客观公正，不偏不倚

---

## 📥 输入要求

### 必须提供
| 输入项 | 说明 | 示例 |
|--------|------|------|
| 会议类型 | 会议的性质 | 董事会/股东会/法务专题会 |
| 会议主题 | 会议讨论的主题 | 并购项目法律风险评估 |
| 会议时间 | 会议召开时间 | 2024年3月15日14:00-16:00 |
| 会议地点 | 会议召开地点 | 公司会议室 |
| 参会人员 | 参加会议的人员 | 董事、法务、财务等 |

### 建议补充
| 输入项 | 说明 | 示例 |
|--------|------|------|
| 会议议程 | 会议讨论的具体议程 | 风险评估、方案讨论 |
| 法律事项 | 涉及的法律决策事项 | 是否提起诉讼 |
| 表决事项 | 需要表决的事项 | 投资方案表决 |
| 决议要求 | 决议的通过要求 | 三分之二以上通过 |

## 🧠 分析框架

### 1. 会议类型分类
| 会议类型 | 适用情形 | 非诉特殊关注点 |
|----------|----------|----------------|
| 股东会 | 公司重大事项决策 | 程序合规、表决有效 |
| 董事会 | 公司经营决策 | 权限范围、程序合规 |
| 监事会 | 监督事项讨论 | 监督权限 |
| 法务专题会 | 法律问题专题讨论 | 法律风险评估 |
| 合规委员会 | 合规事项审议 | 合规要求 |
| 项目评审会 | 项目法律评审 | 项目风险 |

### 2. 会议记录要素
| 记录要素 | 记录内容 | 非诉特殊风险 |
|----------|----------|--------------|
| 程序记录 | 会议召集、主持、表决程序 | 程序瑕疵 |
| 出席记录 | 出席人员、缺席人员 | 出席人数不足 |
| 发言记录 | 主要发言内容 | 遗漏重要意见 |
| 表决记录 | 表决事项、表决结果 | 表决瑕疵 |
| 决议记录 | 决议内容、执行安排 | 决议不明确 |

### 3. 法律风险识别
| 风险点 | 风险内容 | 防控措施 |
|----------|----------|----------|
| 程序瑕疵 | 会议程序不符合规定 | 严格按程序记录 |
| 出席不足 | 出席人数不符合要求 | 准确记录出席情况 |
| 表决瑕疵 | 表决程序或结果有问题 | 准确记录表决过程 |
| 记录遗漏 | 重要内容未记录 | 完整记录 |
| 记录失真 | 记录与实际情况不符 | 客观准确 |

### 4. 不同立场下的记录策略
| 立场 | 核心策略 | 记录特点 | 重点内容 |
|------|----------|----------|----------|
| 记录方立场 | 客观准确、完整记录 | 客观、完整、准确 | 程序、表决、决议 |
| 参会方立场 | 确保己方意见被记录 | 充分表达己方观点 | 己方发言、反对意见 |
| 中立立场 | 平衡记录各方意见 | 公正、全面 | 各方观点、共识分歧 |

## ⚠️ 特别说明

1. **程序合规**：严格按法定/章程程序记录
2. **出席准确**：准确记录出席、缺席情况
3. **发言客观**：客观记录发言内容，避免主观评价
4. **表决准确**：准确记录表决过程和结果
5. **决议明确**：决议内容应明确、可执行
6. **签字确认**：参会人员应签字确认
7. **档案管理**：会议纪要应妥善归档保存
